## Background

> springboot项目开发脚手架

提出公共代码，减少重复性的体力劳动，聚焦业务逻辑实现。主要技术架构如下：

- spring boot 2.0
- spring data jpa
- shiro
- redis
- rabbitMq
- dubbo
- quartz
- [x] queryDsl

## features

1. 角色用户的权限管理
2. 全局异常处理以及状态码
3. 操作日志
4. 任务调度
5. 动态数据源配置
